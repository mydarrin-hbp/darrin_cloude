## 🚀 Deploy pe Vercel

1. Fork sau push acest repo pe GitHub
2. Mergi pe [vercel.com](https://vercel.com) → **New Project**
3. Selectează repo-ul `mydarrin-hbp/darrin_cloude`
4. Framework Preset: **Other** (site static)
5. Root Directory: `/` (root)
6. Click **Deploy** → live în 60 secunde

URL automat: `https://darrin-cloude.vercel.app`
