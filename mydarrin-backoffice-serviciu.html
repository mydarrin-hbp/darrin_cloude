Formula completă — calculată automat de Deviz Engine
1.
COST_BAZĂ = Σ(resurse_MATERIAL + resurse_LABOR + resurse_EQUIPMENT + resurse_TRANSPORT)
2.
COST_NIVEL = COST_BAZĂ × DevizRule.multiplier (BRONZ=1.0 / ARGINT=1.15 / AUR=1.3 / PLATINUM=1.5)
3.
+ COSTURI_INDIRECTE += COST_NIVEL × indirect_cost_percentage (default 10%)
4.
+ MARKETING += marketing_percentage (configurat BO)
5.
+ MENTENANȚĂ += platform_maintenance_percentage (default 3%)
6.
+ COMISION HBP += mydarrin_platform_percentage (7–20% configurabil)
7.
× COEF_ZONĂ = zone_coefficient_override (detectat din IP → localitate → zonă)
8.
× COEF_LEGISLAȚIE = legislation_coefficient (RO-STD=1.0 / EU-STD / MD-STD)
9.
× URGENȚĂ = urgency_coefficient (Normal=1.0 / Urgent=1.3 / Noapte=1.5 / Duminică=2.0)
10.
PREȚ FINAL = toate cele de sus × (1 + vat_percentage) (TVA detectat automat per IP/țară)
